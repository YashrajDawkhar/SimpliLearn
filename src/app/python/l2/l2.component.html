Data Types
  1. Data types are used for define type of Variable
  2. There are 5 types of Data types
    • Numeric
    • String
    • List
    • Tuple
    • Dictionary

1. Numeric :

    # Create variable With integral variable
    a=10
    print("Type of variable is :" ,type(a))

    # Create variable With float variable
    b=34.67
    print("Type of variable is :" ,type(b))

    # Create variable With complex variable
    c=10+8j
    print("Type of variable is :" ,type(c))

Output :

 Type of variable is :  <class 'int'>
 Type of variable is :  <class 'float'>
 Type of variable is :  <class 'complex'>
Note :
  1. int: holds signed integers of non limited length.
  2. long: holds long integer
  3. float: holds floating precision numbers and it's accurate upto 15 decimal places.
  4. complex: holds complex numbers

2. String :

    str1="My"
    str2="String"
    print(str1)
    print(str2)
    print(str1+str2)
    print(str2*3)

Output :

My
String
MyString
StringStringString
Note :
  1. String is a collection of characters
  2. we can use ' ' , " " or ''' ''' to define string
  3. + operator is used to concatenate two string
  4. * operator is used as a repetition operator

3. List :

    # List of only integers
    a=[10,20,30,40]
    print(a)

    # List of only String
    b=['sun','mon','tue','thu']
    print(b)

    # List of both string and integers
    c=[10,'red',65,'blue',54]
    print(c)

Output :

[10, 20, 30, 40]
['sun', 'mon', 'tue', 'thu']
[10, 'red', 65, 'blue', 54]
Note :
  1. It can simultaneously holds different types of data in list
  2. It is ordered sequence of some data written using square brackets([]) and commas(,)

4. Tuple :

    # List of only integers
    a=(10,20,30,40)
    print(a)
                    
    # List of only String
    b=('sun','mon','tue','thu')
    print(b)
                    
    # List of both string and integers
    c=(10,'red',65,'blue',54)
    print(c)

Output :

(10, 20, 30, 40)
('sun', 'mon', 'tue', 'thu')
(10, 'red', 65, 'blue', 54)
Note :
  1. Tuple is a collection of data types
  2. Tuples are sperated by ( , )commas and element enclosed in () parenthesis
  3. Tuple are immutable or read-only: we cannot modify the size and value of tuples
