Reserved Keywords

Reserved Keywords:

  1. The keywords are special words reserved for some purpose.
  2. You cannot use a keword as a variable name.
  3. They are used to define the syntax and structure of python language.

List of Reserved Keywords:

False class finally is return
None continue for lambda try
True def global not while
and del from nonlocal with
as elif if or yield
assert else import pass
break except in raise
Note :
  • Reserved words are case-sensitive and must be used exactly as shown.
  • They are all entirely lowercase, except for False , None and True.
  • You can see reserved keywords list any time by typing help("keywords") to the Python interpreter.
