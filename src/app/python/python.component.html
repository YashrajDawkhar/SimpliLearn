Python is a programming language.

What is Python:

  • Python is Simple , General Purpose , High-Level Programming Language.
  • Python is Object Oriented programming language.
  • Python is case-sensitive language.

Python Features:

  1. Free and Open Source
  2. High-Level Programming Language
  3. Simple and Easy
  4. Portable
  5. Interpreted
  6. Object Oriented programming language
  7. Embedded
  8. Rich set of Functionality
  9. Integrated
  10. Extensible

Explanation :

  1. Free and Open Source :
    Python is free and Open Source Programming Language.The Software can be Downloaded and must be used freely.The Open Source means "Anyone can Download its source code without paying any penny."

  2. High-Level Programming Language:
    Python has been designed to be a high-level programming languages, which means that when you code in Python you don't need to aware of the coding structure, architecture as well as memory management.

  3. Simple and Easy:
    It is Simple and Easy to learn as compared to other programming languages. Its syntax is straightforwad. It is recommended programming language for beginners.

  4. Portable:
    It is portable. That means Python programs can be executed on various platform without altering them.

  5. Interpreted :
    The Python is an Interpreted Language. It means the Python program is executed one line at a time. The advantage of being interpreted language, it makes debugging easy and portable.

  6. Object Oriented programming language:
    Python supports object-oriented language and concepts of classes and objects come into existence. It supports inheritence, polymorphism and encapsulation, etc. The object-oriented procedure helps to programmer to write reusable code and develop application in less code.

  7. Embedded:
    It can be embedded within your C or C++ programs.The code of other programming language can use in the Python source code. We can also use Python Source code in another languages as well.

  8. Rich set of Functionality:
    It is Rich set of functionality available in huge standard library.

  9. Integrated:
    It can be easily integrated with languages like C ,C++ and Java etc. Python run code line by line like C ,C++ and Java. It makes easy to debug the code.

  10. Extensible:
    It implies that other languages like C, C++ can be used to compile the code and thus it can be used further in our Python code. It converts the program into byte code and any platform can use that byte.
