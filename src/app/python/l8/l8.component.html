Selection Statements

Selection / Conditional Branching Statements :

  1. The selection or conditional branching statements are used as decision making statements.
  2. Types of selection statements
    • if statement
    • if-else statement
    • Nested if statement
    • if-elif-else statement

if Statements :

    # Write python program to print even odd number
    n=int(input("Enter value of n : "))
    
    if n%2==0:
       print("Even Number")
    if n%2==2:
       print("Odd Number")

Output:

Enter value of n : 5
Odd Number

Note :
  1. if statement is used to test particular condition.
  2. If the condition is true then it executes the block of statements which is called as if block.
  3. The if statements is simplest form of the conditional statement.
  4. Syntax: 
      if condition:
                   statement

if-else Statements :

    # Write python program to print even odd number
    n=int(input("Enter value of n : "))
    
    if n%2==0:
       print("Even Number")
    else:
       print("Odd Number")

Output:

Enter value of n : 5
Odd Number

Note :
  1. The if-else statement provides an else block combined with the if statement which is executed in the false case of the condition.
  2. If the condition is true then the if-block it executed.
  3. Otherwise , the else-block is executed.
  4. Syntax: 
      if condition:
                   statement
      else:
                   statement

Nested if Statements :

    # Write python program to compare two numbers using nested condition.
    a=int(input("Enter value of a : "))
    b=int(input("Enter value of b : "))
    
    if a==b:
       print("Both the numbers are equal ")
    else:
       if a<b:
          print("a is less than b")
       else:
          if a>b:
             print("a is greater than b")

Output:

Enter value of a : 5
Enter value of b : 4
a is greater than b

Note :
  1. When one if condition is present inside another if then it is called Nested if conditions.
  2. Any nuber of these statements can be nested inside one another.
  3. Indentation is the only way to figure out the level of nesting.
  4. Syntax: 
      if condition:
               statement
      else:
              if condition:
                         statement
              else:
                         if condition:
                                    statement

if-elif-else Statements :

    # Write a python program to display the result such as destinction , 
 first class, second class , pass or fail based on the marks entered by the user 
    n=int(input("Enter your marks : "))
    
    if n>=75:
       print("Grade: Distinction")
    elif n>=60:
       print("Grade: First Class")
    elif n>=50:
       print("Grade: Second Class")
    elif n>=40:
       print("Grade: Pass")
    else:
       print("Grade: Fail")

Output:

Enter your marks : 45
Grade: Pass

Note :
  1. Sometimes there are more than two possibilities. This possibilities can be expressed using chained conditions.
  2. The elif is basically abbreviation of else if.
  3. There is no limit on the number of elif statemets
  4. If there is else clause then it should be at the end
  5. Syntax: 
      if condition:
               statement
      elif condition:
               statement
       ....
      else:
               statement
