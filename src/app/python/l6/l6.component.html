Operators

Operators:

  1. Operators are used to perform operations on variables and values .
  2. For example + , - , * and / are used for performing arithmatic operation.
  3. Python divides the operators in the following groups:
    • Arithmetic Operators
    • Comparison Operators
    • Logical Operators
    • Bitwise Operators
    • Assignment Operators
    • Membership Operators
    • Identity Operators
    • Modulus Operators
    • String Operators

1. Arithmetic Operators:

Arithmetic operators are used with numeric values to perform common mathematical operations

Sr. No. Operators Meaning Example
1 + Addition operator is used for performing addition of two numbers 10+20=30
2 - Subtraction operator is used for performing subtraction 20-10=10
3 * Multiplication operator is used for performing multiplication 20*10=200
4 / Division operator is used for performing division of two numbers 20/10=2
5 % Mod operator returns the remainder value 10%2=0
6 ** This is an exponentiation operator(power) 2**3=8
7 // This is floor division operator. In this operation the result is the quotient in which the digits after the decimal point are removed 10//3=3

6. Membership Operators:

Membership operators are used to find whether a value is a member of a sequence such as or list

Sr.No. Operators Meaning
1 in It returns True if a sequence with specified value is present in the object.
2 not in It returns True if a sequence with specified value is not present in the object.

Examples:

# Membership operators
    color=['red','blue','green']
    print('blue' in color)

    print('yellow' in color)

    print('yellow' not in color)

Output:

True
False
True

7. Identity Operators:

Identity operators are used to determine whether a value is of a certain variable contains.

Sr.No. Operators Meaning
1 is It returns True if both the operand point to point to same memory location.
2 not is It returns True if both the operand point to different memory location.

Example:

    # Identity operators
    color1=['red','blue','green']
    color2=['red','blue','green']
    color3=color1

    print(color1 is color2)

    print(color1 is color3)

    print(color1 not is color2)

Output:

False
True
True
