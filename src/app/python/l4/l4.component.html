Comments

Comments :

  1. Comments are the kind of statements that are written in the program for understanding purpose.
  2. With the help of comment it is possible to understand what exactly the program is about.
  3. Python interpreter ignores comments and will not be visible on screen .

Example :

    # This is a comment line
    print("I love my country")
    # This is  
    # another example 
    # of comment line
    print("Hello India")

Output :

I love my country
Hello India

Note :
  1. We use hash ( # ) to start writing a comment.
  2. For multi-line also we use hash ( # ) in the beginning of each line.
