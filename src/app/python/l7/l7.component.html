Expression

Expression :

  1. Expression is a combination of values, variables and operators.
  2. Value is consider as an expression.
  3. In python operators are special symbols that designate that some sort of computation should be performed.
  4. The value that an operator acts on are called operands.
  5. A sequence of operands and operators , like a+b-5 , is called an expression
  6. python supports many operators for combining data objects into expressions.

Types of expressions :

  • There are three types of Expression
    1. Infix Expression
    2. Prefix Expression
    3. postfix Expression

1.Infix Expression

  • It is a type of expression in which , expression is written in the form as
    • " operand1 operator operand2 "
  • Example:
    a+b is an inflix expression.

2.Prefix Expression

  • It is a type of expression in which , expression is written in the form as
    • " operator operand1 operand2 "
  • Example:
    +ab is an preflix expression.

3.Postfix Expression

  • It is a type of expression in which , expression is written in the form as
    • "operand1 operand2 operator "
  • Example:
    ab+ is an postflix expression.
