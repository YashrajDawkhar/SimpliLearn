Input Operation

1. How input() works in Python

      c=input()
      print("The value of c is : ",c)

Output :

2
The value of c is : 2

Note :
  1. We use input() function to take input from user.
  2. By default input() takes the user's input in a string.

2. Get input from user with a prompt

      X=input("Enter your name : ")
      print("Hello",X)

Output :

Enter your name : Sachin
Hello Sachin

Note :
  1. Here we use input(prompt) function to take input from user.
  2. prompt : It is the string we wish to display on the screen. It is optional.

3. Get input from user using data types :

      a=int(input("Enter First Number : "))
      b=int(input("Enter Second Number : "))
      c=a+b
      print("The value of c is : ",c)

Output :

Enter First Number : 5
Enter Second Number : 4
The value of c is : 9

Note :
  1. To take the input in the form of integer you need to use int() along with input function.
  2. You can also use float , list , tuple along with the input function.
