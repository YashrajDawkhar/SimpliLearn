Installation

Installation :

  1. First Download the latest version of Python from the official Website.
    For downloading Python Click here
  2. Now select the file based on architecture of your PC.
  3. The exectable file Python 3.9.4.exe will get downloaded on your PC.
  4. Go to the where file is downloaded and open the file and just check the Add path checkbox and click Install Now. The Installation will progress
  5. Now finally you will get successful Installation message.
