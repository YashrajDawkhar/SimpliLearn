KeyBoard Shortcut

KeyBoard Shortcut :

  1. The Command Palette :- Ctr + shift + P
  2. Duplicate line up/down :- Shift + Alt + up/down
  3. Move line up/down :- Alt + up/down
  4. Indent or outdent line :- Ctrl + ] or [
  5. Multi-Select :- Ctrl + D
  6. Multi - Select Cursor :- Alt + Click
  7. Toggle line comment :- Ctrl + /
  8. Show / Hide terminal :- Ctrl + `
  9. Toggle side bar :- Ctrl + B
  10. Close Tab :- Ctrl +w
