Basic Tags

Headings :

<!DOCTYPE html>
<html>

  <head>
     <title>Heading Example</title>
  </head>
        
  <body>
        <h1>Heading h1</h1>
        <h2>Heading h2</h2>
        <h3>Heading h3</h3>
        <h4>Heading h4</h4>
        <h5>Heading h5</h5>
        <h6>Heading h6</h6>
  </body>
        
</html>

Output :

Heading h1

Heading h2

Heading h3

Heading h4

Heading h5
Heading h6
Note :
  1. HTML headings are defined with the <h1> to <h6> tags.
  2. <h1> defines the most important heading. <h6> defines the least important heading: 

Paragraphs :

<!DOCTYPE html>
<html>
   <head>
        <title>Paragraph Example</title>
   </head>
   <body>
        <p> first paragraph of text.</p>
        <p> second paragraph of text.</p>
        <p> third paragraph of text.</p>
   </body>
        
</html>

Output :

first paragraph of text.

second paragraph of text.

third paragraph of text.

Note :
  • HTML paragraphs are defined with the <p> tag .
  • Paragraph tag automatically add some space between them .

Line Break :

<!DOCTYPE html>
<html>
   <head>
      <title>Line Break Example</title>
   </head>
   <body>
        <p>Hello<br/>
        I have received the offer letter.<br/>
        Thanks<br/>
        Yashraj</p>
   </body>
        
</html>

Output :

Hello
I have received the offer letter.
Thanks
Yashraj

Note :
  • <br/> tag is used to add break the line .

Horizontal Lines :

<!DOCTYPE html>
<html>
   <head>
        <title>Horizontal Line Example</title>
   </head>
   <body>
        <p>This is paragraph one and should be on top</p>
        <hr/>
        <p>This is paragraph two and should be at bottom</p>
   </body>
        
</html>

Output :

This is paragraph one and should be on top

This is paragraph two and should be at bottom

Note :
  • <hr /> tag is an example of the empty element
  • You do not need opening and closing tags
