Formatting

Formatting :

<!DOCTYPE html>
<html>
   <head>
        <title>Formatting Text Example</title>
   </head>
   <body>
        <b>      1.This text is bold         </b>
        <strong> 2.This text is important!   </strong>
        <i>      3.This text is italic       </i>
        <em>     4.This text is emphasized   </em>
        <small>  5.This is some smaller text </small>
        
        <p>6.Do not forget to buy <mark>milk</mark>today.</p>
        <p>7.My favorite color is <del>blue</del> red.</p>
        <p>8.My favorite color is <ins>red</ins>.</p>
        <p>9.This is <sub>subscripted</sub> text.</p>
        <p>10.This is <sup>superscripted</sup> text.</p>
   </body>
        
</html>

Output :

1.This text is bold
2.This text is important!
3.This text is italic
4.This text is emphasized
5.This is some smaller text

6.Do not forget to buy milktoday.

7.My favorite color is blue red.

8.My favorite color is red.

9.This is subscripted text.

10.This is superscripted text.

Note :
  1. <b> - Bold text
  2. <strong> - Important text
  3. <i> - Italic text
  4. <em> - Emphasized text
  5. <mark> - Marked text
  6. <small> - Smaller text
  7. <del> - Deleted text
  8. <ins> - Inserted text
  9. <sub> - Subscript text
  10. <sup> - Superscript text

Grouping Content :

The <div> and <span> elements allow you to group together several elements to create sections or subsections of a page. 
<!DOCTYPE html>
<html>
   <head>
        <title>Div and span tag Example</title>
   </head>
   <body>
        <div>   Hello i'm div tag    </div>
        <span>  Hello i'm span tag   </span>
   </body>
        
</html>

Output :

Hello i'm div tag
Hello i'm span tag
Note :
  1. div is use for larger part of content and span for smaller part of content
  2. When you create a multiple div tag it will come on next line
  3. When you create a multiple span tag it will come on same line
