Links and Images

Links :

<!DOCTYPE html>
<html>
   <head>
        <title>Hyperlink Example</title>
   </head>
   <body>
        <p>Click any of the following links</p>
        <a href = "https://www.simplilearn.xyz" target = "_blank">Opens in New</a> 
        <a href = "https://www.simplilearn.xyz" target = "_self">Opens in Self</a> 
        <a href = "https://www.simplilearn.xyz" target = "_parent">Opens in Parent</a> 
        <a href = "https://www.simplilearn.xyz" target = "_top">Opens in Body</a>
   </body>
</html>

Output :

Click any of the following links

Opens in New || Opens in Self || Opens in Parent || Opens in Body
Note :

HTML links are defined with the <a> tag:

Sr.No Option & Description
1

_blank

Opens the linked document in a new window or tab.
2

_self

Opens the linked document in the same frame.
3

_parent

Opens the linked document in the parent frame.
4

_top

Opens the linked document in the full body of the window.

Images :

<!DOCTYPE html>
<html>
   <head>
        <title>Image in Webpage</title>
   </head>
   <body>
        <p>Simple Image Insert</p>
        <img src = "SimpliLearnLogo.jpg" alt = "SimpliLearnLogo" />
   </body>
</html>

Output :

Note :

HTML images are defined with the <img> tag.

alt : If img is not viewed it will display message in this case SimpliLearnLogo

Width and Height : It specify the height and width of image

Extra : You can directly add images from internet just copy image address and paste in src .
Also from any folder in computer just right click - properties copy Location and paste in src .

Download Link :

<!DOCTYPE html>
<html>
  <head>
        <title>Download Link Example</title>
  </head>
  <body>
        <a href = "https://www.simplilearn.xyz/page.pdf">Download PDF File</a>
  </body>
</html>

Output :

Download PDF File
Note :
You can create link to make your PDF, or DOC or ZIP files downloadable. This is very simple; you just need to give complete URL of the downloadable file
