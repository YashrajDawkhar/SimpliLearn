JavaScript Home
  • JavaScript is a dynamic computer programming language.
  • JavaScript allow client-side script to interact with the user and make dynamic pages.

Syntax :

<html>
    <body>   
        <script language = "javascript" type = "text/javascript">
        
        document.write("Hello World!");
        
        </script>      
    </body>
</html>

Output :

Hello World!
Note :
  1. JavaScript is written in script tag .
  2. Script tag can be use in head or body part .
  3. But head tag run first so we generally prefer body tag so javascript will run after html load .
  4. document.write is use to print the output .

  • Language − This attribute specifies what scripting language you are using. Typically, its value will be javascript. Although recent versions of HTML (and XHTML, its successor) have phased out the use of this attribute.

  • Type − This attribute is what is now recommended to indicate the scripting language in use and its value should be set to "text/javascript".

What is Framework ?

Frameworks are software that is developed and used by developers to build applications.
Note :
  1. Developing software is a complex process.
  2. It include coding, designing and testing.
  3. Software frameworks make life easier for developers by allowing them to take control of the entire software development process, or most of it, from a single platform.
